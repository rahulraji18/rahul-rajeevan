## Recommended Repository / Folder Structure

This portfolio repo uses a simple structure that can grow into a Next.js app or a static site:

- src/ — Application source
	- src/pages/ — Route pages (for Next.js)
	- src/components/ — Reusable UI components
	- src/styles/ — Global styles, Tailwind config
- public/ — Static assets (images, icons)
- projects/ — Project case studies and demo apps
- scripts/ — Build / deploy helpers
- docs/ — Additional docs, resume, media kit
- .github/workflows/ — CI (build, test, deploy)

Example quick-start files included:

- [index.html](index.html) — simple static landing (starter)
